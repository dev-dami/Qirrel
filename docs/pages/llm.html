LLM Integration

Adapter-driven LLM extensions with resilient config and caching controls.

Providers

  • gemini
  • openai
  • generic

Config

llm:
  enabled: true
  provider: openai
  apiKey: ${QIRREL_LLM_API_KEY}
  model: gpt-4o-mini
  timeout: 30000
  cacheTtl: 300000

Use Adapter

const pipeline = new Pipeline('./config-with-llm.yaml');
await pipeline.init();

const adapter = pipeline.getLLMAdapter();
if (adapter) {
  const response = await adapter.generate('Summarize this text...');
}
