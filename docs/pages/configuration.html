Configuration

Config loading order, sections, and env placeholder behavior.

Loading Order

  • Custom path passed to new Pipeline(configPath)
  • ./miniparse.config.yaml in current working directory
  • Bundled default.yaml

Sample

pipeline:
  enableNormalization: true
  enableCleaning: true
  enableExtraction: true

cache:
  maxEntries: 1000
  ttl: 300000

llm:
  enabled: true
  provider: openai
  apiKey: ${QIRREL_LLM_API_KEY}
  timeout: 30000
  cacheTtl: 300000
Supports placeholders like ${QIRREL_LLM_API_KEY:-fallback}.
