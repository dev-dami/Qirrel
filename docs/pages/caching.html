Caching

LRU + TTL caching at pipeline and component levels.

Config

cache:
  maxEntries: 1000
  ttl: 300000
Set maxEntries: 0 to disable caching.

Manual Access

const pipeline = new Pipeline();

if (pipeline.isCached('input')) {
  const cached = pipeline.getCached('input');
}
