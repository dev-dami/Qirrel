Pipeline Events

Observe run-level and processor-level execution with lightweight handlers.

Events

  • RunStart
  • RunEnd
  • ProcessorStart
  • ProcessorEnd
  • Error

Usage

pipeline.on(PipelineEvent.RunEnd, ({ duration }) => {
  console.log('done in', duration, 'ms');
});
